Resumen

Este proyecto tiene como objetivo desarrollar un sistema de visualizaci√≥n que combine los datos observacionales de la plataforma Global Ocean Data Analysis Project (GLODAPv2.2023) con modelos matem√°ticos simplificados para representar la respiraci√≥n oce√°nica mediante diferentes t√©cnicas, gr√°ficos est√°ticos y animaciones 2D/3D para darle mayor profundidad al analisis, el sistema permite explorar c√≥mo var√≠an par√°metros clave del oc√©ano, como la temperatura, salinidad, ox√≠geno disuelto, nutrientes y carbono inorg√°nico a trav√©s del tiempo y la profundidad.

Introducci√≥n

El oc√©ano desempe√±a un papel importante y fundamental en la regulaci√≥n del clima mediante la absorci√≥n, almacenamiento y liberaci√≥n de gases como el di√≥xido de carbono CO‚ÇÇ y el ox√≠geno (O‚ÇÇ). Este intercambio constante, influenciado por procesos biogeoqu√≠micos, f√≠sicos y biol√≥gicos, se conoce como respiraci√≥n oce√°nica.

Comprenderlo es crucial para evaluar el impacto del cambio clim√°tico, la acidificaci√≥n del oc√©ano y la salud de los ecosistemas marinos. Sin embargo, los procesos involucrados son altamente din√°micos y var√≠an mucho dependiento de la profundidad, la ubicaci√≥n geogr√°fica, la √©poca del a√±o y las condiciones ambientales. Esto hace necesario utilizar herramientas de an√°lisis y visualizaci√≥n que permitan interpretar los datos oceanogr√°ficos de forma clara y accesible y representarlo visualmente. En este trabajo se implemento el dataset GLODAPv2.2023 del Oceano Indico. A partir de los valores observados como la temperatura, salinidad, ox√≠geno, carbono inorg√°nico total, nutrientes y trazadores qu√≠micos.

Se realizaran visualizaciones que representen c√≥mo respira o se comporta la respiraci√≥n en el oc√©ano, mostrando variaciones en gases, perfiles de profundidad, distribuci√≥n espacial y patrones biogeoqu√≠micos. El prop√≥sito general del proyecto es transformar todos estos datos en representaciones visuales intuitivas y √∫tiles tanto para brindar un an√°lisis mucho mas claro.

Agrupaci√≥n y normalizaci√≥n de datos

Para representar perfiles verticales coherentes, las observaciones se agrupan mediante un proceso de binning vertical. Se dividen las profundidades entre 0 y 1000 metros en 20 capas o "bins", lo que reduce la variabilidad extrema producto del muestreo irregular y genera perfiles verticales suaves compatibles con la visualizaci√≥n cient√≠fica tridimensional.

Una vez agrupados los datos, se calcula el promedio por cada capa de profundidad para variables como ox√≠geno, CO2 total, nitrato y temperatura. Posteriormente, estas variables se normalizan mediante la transformaci√≥n min-max, lo que facilita su uso como par√°metros visuales en animaciones (tama√±o de part√≠culas, opacidad, intensidad lum√≠nica, etc.). 


        depth_bins = np.linspace(0, 1000, 20)
        df['depth_bin'] = pd.cut(df['G2depth'], depth_bins, labels=False)

        depth_profiles = df.groupby('depth_bin').agg({
            'G2oxygen': 'mean',
            'G2tco2': 'mean', 
            'G2nitrate': 'mean',
            'G2phosphate': 'mean',
            'G2temperature': 'mean',
            'G2salinity': 'mean',
            'G2depth': 'mean'
        }).reset_index()

        depth_profiles['oxygen_norm'] = (
            depth_profiles['G2oxygen'] - depth_profiles['G2oxygen'].min()
        ) / (
            depth_profiles['G2oxygen'].max() - depth_profiles['G2oxygen'].min()
        )

Animaci√≥n 3D basada en datos reales

En esta etapa se construye una visualizaci√≥n tridimensional que representa la estructura vertical del oc√©ano y su variabilidad simulada a lo largo de un ciclo de 24 horas. El gr√°fico 3D utiliza como ejes la latitud, la longitud y la profundidad, y muestra part√≠culas que representan ox√≠geno, CO2, nutrientes y fitoplancton seg√∫n su concentraci√≥n relativa y posici√≥n en el perfil promedio.

El sistema simula un ciclo d√≠a/noche, durante el cual:

  • Durante el d√≠a aumenta la actividad fotosint√©tica, reduciendo CO2 y aumentando O2
  • Durante la noche predomina la respiraci√≥n, incrementando CO2 y reduciendo O2
  • La visibilidad de part√≠culas biogeoqu√≠micas depende de su profundidad y de la zona f√≥tica
  • La c√°mara 3D rota lentamente para apreciar mejor la estructura volum√©trica

        from matplotlib.animation import FuncAnimation, PillowWriter
        from mpl_toolkits.mplot3d import Axes3D

        def animar_datos_reales(frame):
            ax.clear()
            
            # Configuraci√≥n del espacio 3D
            ax.set_xlim(-5, 5)
            ax.set_ylim(-5, 5)
            ax.set_zlim(0, 1000)

            hora_dia = frame % 24
            
            if 6 <= hora_dia < 18:
                fase = "D√çA"
                intensidad_fotosintesis = max(0, min(1, (hora_dia - 6) / 6))
            else:

                intensidad_fotosintesis = 0
            
            # Visualizaci√≥n de ox√≠geno (burbujas azules)
            oxigeno_data = depth_profiles['oxygen_norm'].values
            profundidades = depth_profiles['G2depth'].values
            
            for i, (prof, oxy) in enumerate(zip(profundidades, oxigeno_data)):

                    num_puntos = int(oxy * 20) + 5
                    x_pts = np.random.uniform(-4, 4, num_puntos)
                    y_pts = np.random.uniform(-4, 4, num_puntos)
                    z_pts = np.full(num_puntos, prof)
                    
                    if fase == "D√çA":
                        size = oxy * 100 + 20
                        color = 'cyan'
                        alpha = 0.7
                    else:
                        size = oxy * 50 + 10
                        color = 'blue'
                        alpha = 0.4
                    
                    ax.scatter(x_pts, y_pts, z_pts, s=size, c=color, alpha=alpha, marker='o')
            
            # Visualizaci√≥n de CO2 (burbujas rojas)
            co2_data = depth_profiles['co2_norm'].values
            
            for i, (prof, co2) in enumerate(zip(profundidades, co2_data)):
                if not np.isnan(co2) and co2 > 0.1:
                    num_puntos = int(co2 * 15) + 3
                    x_pts = np.random.uniform(-3, 3, num_puntos)
                    y_pts = np.random.uniform(-3, 3, num_puntos)

                        size = co2 * 30 + 5
